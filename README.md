# BST/AVL Lab Workspace

This repository is organized into two separate implementations of the same lab assignment.

## Project Layout
- `python_lab/` — Python implementation and report draft.
- `cpp_lab/` — C++ implementation for high-performance full benchmark runs.

## Python Method
See `python_lab/README.md`.

Main files:
- `python_lab/src/trees.py`
- `python_lab/src/benchmark.py`
- `python_lab/LAB_REPORT_RU.md`

## C++ Method
See `cpp_lab/README.md`.

Main files:
- `cpp_lab/src/main.cpp`
- `cpp_lab/build.ps1`
- `cpp_lab/run_full.ps1`
- `cpp_lab/scripts/plot_results.py`

## Notes
- Full benchmark parameters are identical across both methods.
- C++ is recommended for completing full 10x20 runs faster on modern hardware (e.g., Apple M-series).
